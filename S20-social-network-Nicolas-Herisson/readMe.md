# Projet social network

Création d'un site de social network.

# Technos

- Axios
- Sequelize
- PostgreSQL
- Debug
- TypeScript
- Express
- Mongoose
- JSONWebToken
- Crypto

# Objectifs

- Mise en place d'une architecture microservice
- Utilisation de docker (containers, images, volumes, networks, services)
- Utilisation de docker-compose
- Création de services
- Utilisation de Debug
- Utilisation de TypeScript
- Utilisation de Mongoose
- Gestion de l'authentification et l'identification avec JSONWebToken et de Crypto
  
# Setup



## Installer toutes les dépendances

- A la racine du projet:

```bash
pnpm dev:install
```

ou

```bash
npm run dev:install
```

## Setup docker

- Démarrer docker
- A la racine du projet:

```bash
pnpm dev:docker
```

ou

```bash
npm run dev:docker
```
